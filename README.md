# SDMX-DPM Interoperability

This workspace contains documentation and materials for the SDMX-DPM Interoperability work package, exploring how the SDMX standard and the Data Point Model (DPM) can work together across statistical and supervisory domains.

The [concept note](./docs/concept-note.md) of the group stablishes the objectives and deliverables.


## Documentation Structure

The `docs/` folder contains the following sections:

- **index.md** - Home page with workspace overview
- [**concept-note.md**](./docs/concept-note.md) - Work package objectives, scope, and timeline
- [**business-case/**](./docs/business-case/index.md) - Business case for SDMX-DPM interoperability
- [**models-relationships/**](./docs/models-relationships/index.md) - Documentation of alignments between SDMX and DPM models
- [**transformation-guidelines/**](./docs/transformation-guidelines/index.md) - Guidance on SDMXâ†”DPM transformations with examples
- [**background-docs/**](./docs/background-docs/index.md) - Supporting reference materials
- [**meetings/**](./docs/meetings/index.md) - Meeting notes and presentations


## Serving the Documentation

The documentation can be built into html using [MkDocs](https://www.mkdocs.org/). We will explore the possibility to create a web generated automatically from the documentation.
